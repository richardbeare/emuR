Plot superimposed histograms

Usage

hplot(vec, labs, xlab = "", ylab = "", main = "", 
         colour = T, xlim = NULL, axes = T, legn="tl")

Arguments

vec A vector of numerical data.
labs A vector of labels parallel to vec.
xlab An x-axis label.
ylab A y-axis label.
main A main title for the plot.
colour If colour is TRUE, different colours will be used for each unique label. If it is FALSE, different shading patterns will be used.
xlim The range of the x-axis.
axes If FALSE, no axes will be plotted.
legn Where to put the legend. If FALSE no legend will be drawn, otherwise this can be one of "tl", "br", "bl" or "tr" (for top-left, bottom-right etc.) or "loc" (for locator) which requres you to position the legend with the mouse.

Value

None.

Side Effects

Generates multiple overlaid histograms, one for each unique label in labs.

See Also

nplot, eplot
