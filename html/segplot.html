Segment plot

Usage

segplot(segs, param, labels, ylim, ylab = "", main = "", zeros = F, 
         pdat = 0.5, onset = 0, offset = 0, col = T, linetype = F
         smoothing = F, mfrow, addlines, differ )

Arguments

segs An emu segment list.
param One of the valid parameter tracks for the database.
ylim Limit the y-axis range for all plots.
labels A vector of labels to be used as the main title of each plot. If this is not provided the segment labels will be used.
ylab A label for every y-axis.
main A main title for the plot.
zeros If TRUE and there is a prob\_voice track, the regions with probability of voicing less than pdat will not be plotted.
pdat The probability of voicing threshold for zeros.
onset The plot for each segment will start this much to the left of the start of the segment.
offset The plot for each segment will end this much to the right of the end of the segment.
colour If TRUE, different colours will be used in the plot.
linetype If TRUE different linetypes will be used in the plot. Useful for printing.
smoothing If smoothing is TRUE, a non-linear smoother (smooth()) is applied to the data.
mfrow Partitions the plot into sub-plots, eg if there are six segments in the segment list, mfrow might be set to c(2,3). If the mfrow parameter is not supplied, the program estimates a reasonable layout for the plots.
addlines A matrix or vector of time values. If addlines is a vector of time values, then a vertical line is drawn on plot j at the time given in addlines[j]. If addlines is a matrix, then a series of vertical lines are drawn on plot j, for the times given in addlines[j,i] for each column i.
differ An integer, carries out nth order differentiation where n is the value of differ. Defaults to no differentiation.

Side Effects

If overlay is FALSE, one subplot is made per segment and the subplots are arranged in rows, each labelled with the segment label. If overlay is TRUE, one large plot is made and the data for all segments is overlaid. In this case the x-axis is not labelled since each segment will have a different start time and duration.

See Also

emu.query, emu.track, uttplot, dplot

Examples


## assumes a database called demo is available on your system and that
## the Emu system is installed. 

# find all Phonetic vowels in the database
segs <- emu.query("demo", "*", "Phonetic=E")

segplot( segs[1:10,], "F0" )
