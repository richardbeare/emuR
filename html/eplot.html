Ellipse plot

Usage

function(data, labs, chars, formant = F, scaling = "linear", 
         nsdev = 2.447747, dopoints = F, classify = F, 
         doellipse=F, centroid = F, main="", xlab="", ylab="", axes = T,
         xlim, ylim, colour, linetype, ...)

Arguments

data A two-columned matrix of data
labs A vector of labels, parallel to data.
chars A vector of labels, parallel to data. If this argument is specified these labels will be plotted rather than the labels in labs.
formant If TRUE, then the data is negated so that, for formant data, the plot is made in the -F2/-F1 plane. This gives a similar layout to the common vowel quadrilateral.
scaling Either "mel" or "bark" for mel or bark scaling of the data.
nsdev Defines the length of the major and minor axes of the ellipses in terms of the standard deviation of the data.
dopoints If TRUE, character labels (from labs or chars) are plotted for each data point.
doellipse If T, ellipses are drawn on the plot. If F, no ellipses are drawn and, if 'dopoints' is also F, 'centroids' is set to T.
centroid If T, a single character label is plotted at the center of each ellipse.
classify If classify is TRUE, a gaussian (mahalanobis distance) classification is carried out on the data and the confusion matrix is returned.
main A main title for the plot.
xlab, Titles for the x and y axes.
axes If TRUE axes are drawn on the plot.
xlim, The bounds of the plot on the x and y axes, each should be a two element vector giving the minimum and maximum values to plot.
colour If colour is TRUE, the ellipses and labels will be plotted in different colours.
linetype If linetype is TRUE, the ellipses will be plotted with different linetypes. This is useful for plots that will be printed.
This function will also accept any of the standard graphical parameters which can be passed to par(), for example cex which specifies the size of characters to be used.

Value

A confusion matrix, if the classify argument is TRUE, otherwise NULL.

Side Effects

A plot is generated consisting of an ellipse defining the distribution of each class within the given labels. The mean and covariance of each class is estimated from the data and plotted. Labels may be plotted for each data point or at the class mean for each class.

See Also

train(), mahal()

Examples

eplot(formantdata, vowellabs, centroids=T)
