Extract times and durations from an Emu segment list

Usage

mustart(segs)

Arguments

segs An Emu segment list.

Value

A vector containing one element per segment in the segment list. mustart muend extract the start and end times of each segment in a segment list. mudur extracts the duration of each segment.

References

See the Emu documentation at: http://www.shlrc.mq.edu.au/emu

See Also

emu.query

Examples

## assumes a database called demo is available on your system and that
## the Emu system is installed. 

# find all Phonetic vowels in the database
segs <- emu.query("demo", "*", "Phonetic=vowel")

# plot a histogram of segment durations
durations <- mudur( segs )
hplot( durations, label(segs )
