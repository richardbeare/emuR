Translate labels in a label vector

Usage

label.convert(labels, l1, l2)

Arguments

labels A vector of labels. a
l1 The label to be replaced.
l2 The substitute label.

Value

The labels vector with all instances of l1 replaced by l2.

Examples

> label.convert(c("A", "E", "I", "A"), "A", "V")
[1] "V" "E" "I" "V"
