Plot normal data curve

Usage

nplot(values, labs, xlab="", ylab="probability density", 
      main="", colour=T, linetype=F, plot.lab=T, 
      xlim, logprob=F)

Arguments

values A vector of numeric values.
labs A vector of label, parallel to values.
xlab An y-axis label
ylab A y-axis label
main A main title for the plot.
colour If colour is TRUE, the curves and labels will be plotted in different colours.
linetype If linetype is TRUE, the curves will be plotted with different linetypes. This is useful for plots that will be printed.
plot.lab If plot.lab is TRUE, labels will be plotted above each normal curve.
xlim A vector of the minimum and maximum values to plot on the x-axis.
logprob If TRUE, the log-normal curve will be plotted.

Details

The data is split according to the unique labels in labs and the mean and variance of each data set is calculated. The probability density for each data set is then plotted.

Side Effects

Generates a plot on the current device which shows the distribution of the data in values for each unique label in labs.
