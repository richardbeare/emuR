Emu/Splus Help

These help pages document the Splus functions provided with the Emu/Splus package. Some functions are not documented, they are listed at the end of this page.

as.trackdata Create an Emu trackdata object
dextract.lab Extract data from a trackdata
dplot Plot Emu Trackdata
dsmooth Smooth the data in a trackdata object.
ellipse Calculate ellipse coordinates
emu.query Query an Emu Database
emu.track Retrieve Numerical Data
eplot Ellipse plot
euclidean Find the inter-euclidean distance for a data matrix
expand.labels Label each data sample
frames.time Find the time and position of a data element.
get.time.element Get data for a given time
hamming Generate a Hamming window
hplot Plot superimposed histograms
is.segs Test whether an object is a segment list
label.convert Translate labels in a label vector
label.num Convert labels to integers
linear Perform linear time normalisation on trackdata.
mahal Classify using Mahalanobis distance
mahal.dist Calculate mahalanobis distances
matscan Read matrix data from a file
mel Convert Hz to the mel scale
muclass Find common elements in vectors
mudur Extract times and durations from an Emu segment list
museg.write Write an Emu segment list to a file
norm Normalise speech data
nplot Plot normal data curve
percent Converts a confusion matrix to a percentage matrix
perform Performance (hit rate) of a confusion matrix
radians Converts degrees to radians
randomise.segs Randomise or Reverse items in a segment list
segplot Segment plot
slen Number of Emu Segments
sortmatrix Sort matrix by label
splitmat Split data according to a set of labels
splitstring Split a string into words.
steady Find the steady-state portion of a parameter
train Train a Gaussian Model
utt Return the utterances portion of a segment list
uttnum Find an utterance in Utterances
uttplot Plot a parameter for an utterance
