Extract a subset of data from a trackdata object

Usage

dextract.lab(trackdata, labs, labtype=unique(labs))

Arguments

trackdata A trackdata object returned from track.
labs A vector of labels parallel to trackdata$index, i.e. one for each segment in the trackdata.
labtype A vector of labels for which data is to be extracted.

Value

A trackdata object which is a subset of trackdata containing only the data for those labels in labtype. The result has the same components as the input trackdata:

data A vector or matrix of numerical data.
index A two column matrix giving the start and end indeces into the data vector for each segment.
ftime A two column matrix giving the start and end times for each segment.

See Also

track, dextract, get.time.element, frames.time

Examples

# extract only the data for i: and I vowels
library(emu)
# get some vowel segments and the corresponding formant data
vsegs <- emu.query("demo", "*", "Phonetic=vowel")
vdata <- emu.track(vsegs, "fm")
# extract just the data for the e: and E vowels
i.data <- dextract.lab(vdata, label(vsegs), c("e:", "E"))
# generate a plot of the data
dplot(i.data, main="e: and E vowels", average=T, normalise=T)
